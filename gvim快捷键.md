# GVim快捷键
|作用|指令|
| ------ | ------ | 
| 打开文件目录树 | :Vexplore | 
| 稍微长一点的文本 | 短文本 | 
|搜索全词匹配|/\\<int\\> |
|忽略大小写|:set ignorecase|
|大小写敏感|:set noignorecase|
|关闭当前编辑文档打开另一个文档|:e + 文件路径|
|打开一个新的标签页|:tabe|
|显示已打开标签页的列表|:tabs|
|移动到第n个标签页|:tabm n|
|全局替换|: %s/string1/string2/g|
|全局替换且一一确定|: %s/string1/string2/c|
|筛选关键字的行|:v/string/d|
